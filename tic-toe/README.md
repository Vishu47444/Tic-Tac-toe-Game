1. Navigate to the `tic-toe` folder
2. Double-click on `index.html` to open it in your default browser
3. Start playing!

### Option 3: Local HTTP Server
```bash
cd tic-toe
python -m http.server 8000
# Then open http://localhost:8000 in your browser
```
